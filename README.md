# Smart Parking Management System

A web-based Smart Parking Management System developed using HTML, CSS, and Vanilla JavaScript. The project provides an interactive interface to manage parking slots and visualize parking availability.

## Live Demo

https://k2adarsh.github.io/Smart-Parking-Management-System/

## Features

- Interactive parking slot management
- Displays available and occupied parking spaces
- Dynamic updates using JavaScript
- Clean and responsive user interface
- Simple browser-based implementation

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
